# Dispensa Mic-1

Dispensa didattica sul processore Mic-1 per il corso di Architettura dei Sistemi
di Elaborazione.

## Contenuti

* [Requisiti](#requisiti)

## Requisiti

* Una [distribuzione TeX](https://www.latex-project.org/get/)
* CMake

## Build

```sh
$ mkdir build && cd build && cmake .. && make
```
